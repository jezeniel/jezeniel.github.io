# Personal Site

Personal website made with :heart: using [pelican](https://blog.getpelican.com/) with
[end2end](https://github.com/nandomoreirame/end2end) as the base theme.

## Installation

```bash
$ pip install -r requirements.txt
$ npm install
```

## Run development server

```bash
$ make devserver
```

## Watch CSS 

```bash
$ npm run watch
```
